
# This Document details when Pressing the Respective Switches, Which wires will be shorted to Indicate the user has pressed a Particular switch.
        Left Side
		
		Wire Colour                 -> 		utility
		Black + Black-Green 		-> 		Horn
		Yellow + White 				-> 		Headlights and Taillights
		Blue + Yellow  				->		HighBeam (Including Pass)
		Light Blue + grey			-> 		Right Indicator
		Red + grey    				-> 		Left Indicator
		
		
		Right Side
		
		Wire Colour                 -> 		utility						-> 		Motor Controller Pin
		Red-Black	+ Red	      	->		Igniton Switches
		Red-Yellow	+ Red	        ->		Boost Mode
